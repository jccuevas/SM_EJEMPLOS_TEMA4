Tema 1

Objetivos

  1. Conocer los orígenes y características más significativas de Android.
  2. Requerimientos mínimos del terminar para ejecutar Android.
  3. Conocer las diferentes versiones de Android.
  4. Conocer la información disponible de Android.
  5. Conocer los distintos métodos para almacenar datos de manera persistente en Android.
  6. Conocer la problemática del almacenamiento físico en Android.
  7. Conocer el uso de las bases de datos en Android.

Contenido

El el Tema 1 se verá una breve introducción a Android, así como los principios básicos para la creación de aplicaciones para este sistema operativo.

Aplicación

La aplicación cuenta con numerosos ejemplos para ilustrar los contenidos a lo largo del Tema 1.
